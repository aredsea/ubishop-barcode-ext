### 2. 작업 C Phase 2 — ERP 디코드 SSOT 이관 (`908c237`, app v2.9.1)
Phase 1 의 `src/erp.js` 가 **아무도 안 부르던 상태**였는데 MAIN world 3파일이 이제 호출한다.

**★핵심 판단 — 헤더 정책을 통일하지 않고 사이트별로 보존했다. 다음 세션도 이 원칙을 지켜라.**
`decodeErpHtml(bytes, contentType)` 은 charset 선언을 **신뢰**한다. 서버가 오선언하면 그대로 깨지고
이건 의도된 동작으로 `tests/erp-decode.test.js:53-55` 에 문서화돼 있다. 따라서 **헤더를 안 보던
곳에 contentType 을 넘기면 오늘 잘 되던 응답이 깨진다.**

| 파일 | 호출 | 근거 |
|---|---|---|
| `collector.js` | `decodeErpHtml(buf)` 1인자 | 기존 `decodeKr`(score-both)와 **바이트 동일** — `erp-decode` `[f]` differential 이 증명. 무회귀 |
| `statis.js` | `decodeErpHtml(buf)` 1인자 | 헤더 정책 유지. threshold-20 → score-both 로 개선 |
| `cache-intercept.js` | `decodeErpHtml(buf, ct)` **2인자** | 원래 헤더 선언을 신뢰하던 곳이라 정책 보존 |

이 1인자/2인자 구분은 `tests/erp-wiring.test.js` 가 구조적으로 고정한다(바꾸면 실패).
부수: `cache-intercept` 진단필드 `decoder` 는 채택 인코딩을 알 수 없어 `'ubErp'` 로 표기
(읽는 소비자 0건, `contentType` 은 유지 — 가짜 정밀도 회피).

### 4. `loader-integrity` 강화 (같은 커밋)
번들이 7파일이 되며 `assert(files.length === 6)` 가 깨졌다. **완화하지 않고 강화**했다.
- 개수 상수 → **순서 있는 구성 목록 전체 고정**(배포 계약 자체를 고정)
- `files ↔ sha256` **키 집합 일치** 추가 — 기존엔 아예 없던 검사(제거된 파일의 잔재 해시를 못 잡았음)

> 경위: 처음엔 "개수 상수는 의식이지 검증이 아니다"라며 개수를 빼고 집합일치만 뒀는데,
> **Codex 가 구멍을 실증**했다 — `files` 와 `sha256` **양쪽에서 같이** 지우면 통과한다.
> 개수를 뺀 판단은 맞았지만 **구성 불변식을 대체 없이 버린 게 실수**였다. 목록 고정으로 바로잡음.

## 이 세션에서 새로 당한 함정 (추가)
- **`grep -c $'\r'` 가 CR 개수를 거짓 보고한다.** 패턴이 빈 문자열로 무너져 **전 줄에 매치**한다
  (LF 전용 파일이 "139줄에 CR 있음"으로 나왔다). **바이트 세는 건 node 로** 해라.
  → 저장소 기존 함정 "Bash heredoc 이 백슬래시를 삼킨다"와 같은 뿌리.
- **인덱스 해시를 작업트리 raw 바이트로 비교하면 "스테일" 오진이 난다.** `core.autocrlf=true` 라
  일부 파일이 CRLF 로 체크아웃되는데, 인덱스 해시는 **LF 정규화 기준**(= GitHub raw 가 서빙하는 바이트).
  `build-app-index.js` 는 이미 LF 정규화 후 해시하고 단독 CR 이면 멈춘다 — **툴은 정상이다.**
  검증할 땐 LF 정규화하거나 `git show HEAD:<path>` 로 커밋된 blob 을 써라.
- **`statis.js` 의 `render` 동명이인은 아직 남아 있다.** 죽은 것 하나는 지웠지만 여전히
  `renderOrder` 안 **중첩 `render()`** 와 `skin.js` 의 `render(...)` 가 있다. 소스를 슬라이스하는
  테스트는 **`renderOrder` 본문 기준으로 앵커**를 잡아라(`tests/statis-xlsx.test.js:41` 주석 참고).
- **테스트에 "죽은 코드가 존재한다"는 단언을 두지 마라.** 죽은 코드 제거 작업을 막는 지뢰가 된다
  (실제로 초안에 있어서 뽑았다).
