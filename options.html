🏷 유비샵 바코드 라벨 인쇄

① 라벨 위치/크기 조정

유비샵 상품검색 페이지 우하단의 🏷 라벨위치 버튼을 누르세요. (이 옵션 페이지가 아니라 유비샵 페이지 안에서 합니다.)

  • 이동: 항목을 드래그 / 화살표키(0.5mm, Shift 0.1mm)
  • 크기: 선택 항목 우하단 파란 핸들을 대각선으로 드래그 (글자=글자크기, 바코드/박스=가로·세로)
  • 다 맞췄으면 저장 → 이후 인쇄에 바로 적용됩니다.

처음 한 번은 페이지를 열면 편집기가 자동으로 뜹니다. 맞추고 저장하세요.

② 인쇄창 없이 자동 인쇄

크롬은 확장만으로 인쇄창을 못 없앱니다. 확장 폴더의 자동인쇄-설정.bat(또는 autoprint-setup.vbs)을 1회 더블클릭하면, 이후엔 평소처럼 크롬을 켜기만 해도 인쇄창 없이 바로 인쇄됩니다.

  • 설정 후 크롬을 모두 닫았다가 다시 여세요.
  • Windows 기본 프린터 = Zebra 인지 확인하세요.

자동업데이트: 코드가 바뀌면 페이지를 새로고침하면 최신으로 적용됩니다(인터넷 연결 시).
